## Descrição do Projeto

Este projeto foi desenvolvido para a disciplina de Mobile utilizando **React Native com Expo**.  
O aplicativo consiste em um **formulário de cadastro simples**, onde o usuário pode preencher suas informações e visualizar os dados enviados na própria tela.

## Funcionalidades

- Campo para preencher o **nome**
- Campo para preencher o **curso**
- Campo para preencher a **disciplina**
- Campo para preencher uma **descrição pessoal**
- Botão para envio do formulário
- Exibição dos dados preenchidos na própria tela
- Validação para impedir envio com campos vazios
- Uso de `useState` para controle dos campos
- Uso de `useEffect` ao iniciar o aplicativo
- Estilização personalizada com `StyleSheet`

## Tecnologias Utilizadas

- React Native
- Expo
- JavaScript
- VS Code

## Estrutura do Projeto

O projeto foi desenvolvido de forma simples, centralizando toda a lógica principal no arquivo `App.js`.
